Robot Searching Webpage

React

Search your robots and create new ones! Let your imagination go wild!

This is a project created with Create React App, featuring a real time search result while you type. There is also a 'NEW' section on the top right corner, where you could customize your own robot, preview its profile, and add to your list if you want! More over, if you don't have a clear idea in mind for your robot, you can use the 'random' button to generate a robot automatically!
Note that there is no database in this project, so any changes made on the list would not be saved.
